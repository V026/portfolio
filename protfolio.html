Portfolio Website


Project Overview

Designed and developed a modern personal portfolio website to showcase professional experience, projects, education, skills and contact details.

Features

  • Modern dark UI design
  • Responsive website layout
  • Project cards with navigation
  • Contact form integration using FormSubmit
  • GitHub Pages deployment
  • Social media integration
  • Resume download option
  • Animated effects

Technologies Used

  • HTML5
  • CSS3
  • JavaScript
  • GitHub Pages
  • FormSubmit API

Key Learnings

  • Frontend development workflow
  • Responsive design techniques
  • Git and GitHub deployment
  • Form handling and hosting
  • UI/UX design practices
